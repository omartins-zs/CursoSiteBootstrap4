⚙ Instalação o Bootstrap 4 (com Sass e Gulp)
⚗ Transformação do layout em um tema de WordPress
✏ Criação das funcionalidades do tema
🕶 Utilização de plugins úteis
📡 Publicação do site

npm i bootstrap@4.6.2

npm install jquery

npm install popper

npm install -g sass

sass --version

sass --watch node_modules/bootstrap/scss:node_modules/bootstrap/compiler

npm install font-awesome

sass --watch node_modules/bootstrap/scss:node_modules/bootstrap/compiler node_modules/font-awesome/scss:node_modules/font-awesome/css style/scss:style/css
